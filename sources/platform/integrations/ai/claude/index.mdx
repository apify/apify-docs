Apify integrates with [Anthropic Claude](https://www.anthropic.com/claude) in two places: the Claude Desktop app (via the Apify MCP server) and the Claude Code CLI (via the Apify plugin). Both give Claude a way to call Apify Actors as tools and read their results.
