[Codex](https://developers.openai.com/codex/) is OpenAI's agentic coding tool. The [Apify plugin for Codex](https://github.com/apify/apify-codex-plugin) connects Codex to Apify's library of [Actors](https://apify.com/store) through the [Apify MCP server](/platform/integrations/mcp) and five bundled skills for common workflows.

The plugin installs in two places. Pick the one that matches how you use Codex.
